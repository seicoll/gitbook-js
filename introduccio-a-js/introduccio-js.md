# Introducció a JavaScript

## Què és JavaScript?

> **Javascript** és un llenguatge de programació que s'executa principalment al navegador.

- Permet crear pàgines web interactives.
- Javascript és **interpretat** directament pel navegador.

## Història

- **Brendan Eich**, treballant a Netscape, crea **LiveScript** al **Netscape Navigator 2.0**
  - Després canvia el nom a **JavaScript**.

![Brendan Eich](https://upload.wikimedia.org/wikipedia/commons/0/09/BEich.jpg)

- **Microsoft** llança **JScript** amb **Internet Explorer 3** (1996).

  - És una implementació molt similar a JavaScript, amb un nom diferent per evitar problemes legals amb Netscape.

- **Netscape** proposa l’estandardització del llenguatge a l’organisme **ECMA International**.

  - El **1997**, ECMA publica el primer estàndard oficial amb el nom **ECMAScript (ES1)**.

- El **juny de 2015** es publica **ECMAScript 6**, també conegut com **ECMAScript 2015 (ES6)**, una versió clau que introdueix grans millores com `let`, `const`, classes, mòduls i funcions fletxa.

- A partir de **2016**, ECMAScript adopta un **model de versions anuals**, amb noms basats en l’any de publicació:

  - **ECMAScript 2016 (ES7)**
  - **ECMAScript 2017 (ES8)**
  - **ECMAScript 2018 (ES9)**
  - ...

- **Actualment**, ECMAScript **no té una “última versió única”**, ja que es publica **una nova edició cada any**.
  - Les versions recents (com **ES2022, ES2023, ES2024**) introdueixen millores incrementals del llenguatge.
